# Proyecto Diplomado

### 1. Clonar el Proyecto
```bash
git clone https://github.com/blady24/diplomado.git
```

### 2. Instalar Dependencias
```bash
npm install
```

### 3. Copiar Variables de entorno y configurar
```bash
cp .env.sample env
```

### 4. Levantar el Proyecto
#### 4.1 En desarrollo
```bash
npm run dev
```

#### 4.2 En produccion
```bash
npm run star
```
